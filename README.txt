Building the Game:-

use make carrom MODE=a FRICTION=b EREST=c to generate the executable.
Here,
1) a= 1 for playing as server, 2 for playing as client , 3 for offline play.
2) b can have have any value from 1-10 in increasing order of friction level in game.
3) c defines the coefficient of restitution with value between 0-1.

Use ./carrom to run the game.

For more information regarding the design and gameplay options see the carrom_doc.pdf file.
